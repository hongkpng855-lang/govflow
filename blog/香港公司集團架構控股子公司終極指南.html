1. 我點解要寫呢篇文

如果你開咗香港有限公司一段時間，業務開始穩定，甚至開咗第二個生意方向，你可能會諗：「應唔應該開多一間公司嚟做新業務？」

或者你見到身邊啲朋友搞集團公司、控股公司，好似好威好專業，但你又唔知實際上係咩一回事。

我當年就係咁。

開第一間公司嗰陣乜都唔識，得一個生意。後來擴展到第二個業務，我順手就開多一間公司，用同一個股東名單。結果幾年後先知，原來我可以 set 控股公司架構，可以更有效地保護資產、慳稅、甚至將來賣盤都容易啲。

⚠️ 真實經驗：我當初冇 set 集團架構，結果其中一間公司被人告，另一間公司嘅資產理論上都有風險——雖然係獨立法律實體，但同一班股東嘅情況下，銀行同供應商睇你成個 group 嘅信譽，一間出事會影響晒。

呢篇文我會用最白話嘅方式，由基本概念講起，到實際點樣搭集團架構，再講稅務安排。目標係令你以後聽到「集團公司」呢四個字唔會覺得高深，反而知道點樣運用。

2. 基本概念：控股公司、子公司、關聯公司

先搞清楚最基本嘅用詞，唔好俾人嚇親。

控股公司（Holding Company）

控股公司嘅唯一目的就係「持有其他公司嘅股份」。佢本身可以冇業務、冇收入、冇員工，就係手揸住其他公司嘅股票。控股公司通常持有子公司超過 50% 嘅股權，咁先有「控制權」。

例子：你開咗一間公司叫「A Holdings Limited」，呢間公司乜都唔做，淨係持有「B 貿易有限公司」同「C 科技有限公司」各 100% 股份。A Holdings 就係控股公司，B 同 C 就係子公司。

子公司（Subsidiary）

子公司係被控股公司控制嘅公司。控股公司持股超過 50% 就叫「附屬公司」；持股 100% 就叫「全資附屬公司」（Wholly-owned Subsidiary）。

最重要嘅一點：每間子公司都係獨立嘅法律實體。佢哋各自有自己嘅銀行戶口、自己嘅合同、自己嘅債務。子公司欠債，原則上債主追唔到控股公司或者其他子公司。

關聯公司（Associated Company / Sister Company）

關聯公司係指由同一班股東/董事控制嘅多間公司，但彼此之間冇母公司-子公司嘅股權關係。即係同一班人開咗 A 公司同 B 公司，但 A 唔持有 B 股份，B 亦唔持有 A 股份。

例子：你個人持有「A 餐飲有限公司」同「B 零售有限公司」各 100% 股份。A 同 B 就係關聯公司，因為你係共同股東，但 A 同 B 之間冇互相持股。

💡 快速記憶：
• 控股公司 → 持有其他公司股份嗰間（上面嗰間）
• 子公司 → 被持有股份嗰間（下面嗰間）
• 關聯公司 → 同一老細但冇互相持股嘅公司（打橫嘅關係）

3. 點解要 set 集團架構？3 個真實好處

好處 1：資產保護（最重要）

呢個係最大嘅好處。如果你得 1 間公司，所有業務、資產、風險都喺埋一齊。萬一其中一個業務出事（例如被人告、供應商追數、租約糾紛），你成間公司嘅資產都會受影響。

但如果唔同業務分開子公司，A 子公司出事，B 子公司嘅資產係獨立嘅，完全唔會俾人凍結或追討。

✅ 真實案例：我有個朋友做飲食業，一間公司做餐廳（高風險），另一間公司 hold 緊佢嘅物業（低風險）。後來餐廳被人告食物中毒，好在物業嗰間公司完全冇事，因為係獨立子公司。

好處 2：將來的靈活性（賣盤/集資）

如果你將來想賣其中一個業務，或者俾新股東入股，有集團架構會方便好多。你可以直接賣子公司嘅股份，完全唔影響其他業務。如果得 1 間公司，你要抽起某個業務嘅資產，手續會複雜好多。

例子：你有 App 開發業務同顧問業務。有間大公司想收購你嘅 App 業務。如果分咗子公司，直接賣子公司股份就得。如果冇分，你要「業務轉讓」（Business Transfer），要逐個 contract 轉名、通知客戶，麻煩好多。

好處 3：稅務規劃空間

集團架構可以讓你更靈活安排利潤。例如：一間子公司賺錢、另一間蝕錢，集團可以考慮利潤轉移嘅安排。控股公司可以收子公司嘅股息，而香港冇股息稅。持有海外業務嘅控股公司仲可以用外地收入豁免徵稅（FSIE）制度。

⚠️ 注意：稅務安排一定要有商業實質（Commercial Substance），唔可以純粹為咗避稅而 set 架構。稅局會睇你有冇實際營運、有冇員工、有冇辦公室。否則會被視為濫用安排，追討返稅款。

4. 常見集團架構模式

以下係香港中小企最常見嘅幾種集團架構，你可以根據自己情況揀適合嘅：

模式一：一間控股 + 多間營運子公司（最推薦）

你（個人股東）
⬇ 100%
控股有限公司（Holding Company）
⬇ 100%
子公司 A（餐飲）
子公司 B（物業）
子公司 C（電商）

適合：有多個業務方向、想將來賣盤、或者想保護資產嘅東主。

模式二：同一股東持有多間關聯公司（最簡單）

你（個人股東）
⬇ 100%
公司 A（業務 1）
── 關聯關係 ──
公司 B（業務 2）

適合：業務簡單、唔諗住賣盤、想慳成本嘅創業初期。注意：冇控股公司嘅話，將來要賣一間公司，要你自己個人賣股，稅務上可能冇咁著數。

模式三：控股公司 + 海外子公司（跨境集團）

你（個人股東）
⬇ 100%
香港控股公司
⬇
香港營運公司
海外子公司（例如 BVI / Singapore）

適合：有跨境業務、海外客戶或供應鏈，想利用香港嘅 FSIE 稅務優惠。

5. 集團架構下的稅務安排

呢 part 比較 technical，我會盡量講得白啲。

股息上繳（Dividend Upstream）

子公司賺咗錢，可以派股息俾控股公司。香港冇股息稅，所以控股公司收到股息係免稅嘅。控股公司再將資金分配俾其他子公司，或者用嚟投資新業務。

實例：子公司 A 賺咗 HK$100 萬，派息 HK$80 萬俾控股公司。控股公司收到 HK$80 萬完全唔使繳稅，然後可以將呢筆錢注入子公司 B 做擴展資金。

集團虧損抵銷（Group Relief）

香港目前冇正式嘅「集團虧損寬免」制度，即係一間子公司嘅虧損通常唔可以直接抵銷另一間子公司嘅利潤。但可以透過以下方式安排：

  • 集團內部借貸：賺錢嘅公司借錢俾蝕錢嘅公司，利息收入可以平衡利潤
  • 服務費安排：控股公司向子公司收取管理費，將利潤由賺錢公司轉移到控股公司
  • 成本分攤：共同成本（如辦公室、IT 系統）按合理比例分攤
⚠️ 重要提醒：所有集團內部交易都要符合「公平原則」（Arm's Length Principle），即係定價要同市場價格一致。稅局好重視轉讓定價（Transfer Pricing），如果發現集團內部交易價格唔合理，可以調整稅款兼罰款。

外地收入豁免徵稅（FSIE）

2023 年開始，香港實施咗 FSIE 制度。簡單講：控股公司收到嘅境外股息、利息、處置收益，如果符合特定條件（有經濟實質、有足夠員工、有營運開支），可以豁免繳利得稅。

呢個對於有海外子公司嘅集團架構嚟講好重要，因為可以將海外利潤匯返香港而唔使交稅。

6. 由一間公司變集團嘅 Step-by-step

如果你目前得 1 間公司，想 set 集團架構，以下係大致步驟：

1 成立控股公司 — 開一間新有限公司做控股公司，股東同你現有公司嘅股東一致。最好用同一班股東同董事。

2 將現有公司股份轉俾控股公司 — 你將個人持有嘅現有公司股份，轉讓俾控股公司。呢步涉及股份轉讓（Share Transfer），需要簽 Instrument of Transfer 同俾印花稅。

3 更新公司紀錄 — 現有公司嘅股東名冊要更新，顯示控股公司係新股東。董事會要通過決議確認股份轉讓。

4 開新業務用子公司 — 以後開新業務，直接由控股公司出資成立新嘅全資子公司，唔再用自己個人名義開公司。

5 整理內部協議 — 控股公司同子公司之間要有服務協議、管理費協議等，確保有商業實質，稅局唔會質疑。

💡 貼士：Step 2 嘅股份轉讓要俾印花稅（轉讓代價嘅 0.2%），所以最好趁公司股份價值低嘅時候做。如果公司已經好值錢（例如有物業或大量現金），印花稅會好金，建議先搵會計師計清楚。

7. 常見錯誤（我親身見過嘅）

❌ 錯誤 1：以為開咗控股公司就可以自動保護資產

控股公司架構唔係「自動保護」機制。如果子公司簽約嗰陣，對方要求控股公司做擔保人（Personal Guarantee），咁控股公司都要負責。好多銀行貸款都會要求控股公司做擔保，咁就冇咗資產保護嘅作用。

❌ 錯誤 2：控股公司冇商業實質

控股公司淨係得個名，冇辦公室、冇員工、冇銀行戶口、冇董事會議。咁樣稅局唔會承認呢個集團架構，FSIE 免稅用唔到，銀行開戶都難。控股公司最起碼要有：銀行戶口、董事、registered address、每年做返啲董事會會議紀錄。

❌ 錯誤 3：集團內部交易亂咁定價

「子公司 A 收子公司 B HK$100 萬顧問費，但其實係同一班人做」— 稅局一查就知有問題。所有集團內部交易一定要有合約、有合理定價、有實際服務。否則會被視為「利潤轉移」，追討稅款。

❌ 錯誤 4：忘記申報關聯公司資料

如果你有多間關聯公司，喺稅務申報時要披露關聯交易。公司註冊處嘅重要控制人登記冊（SCR）都要記錄每一間公司嘅「重大控制人」。忽略呢啲申報要求，可能會被罰款。

❌ 錯誤 5：為慳稅而 set 集團架構，但冇實際商業需要

稅局唔傻。如果你 set 集團架構純粹為咗將利潤由一間公司搬到另一間，而冇實際商業原因（例如：業務多元化、風險管理、將來賣盤），稅局可以無視你嘅集團安排，直接按實際情況評稅。

8. 咩情況下唔需要 set 集團架構

集團架構唔係適合所有人。以下情況你可能唔需要：

🔍 你得一個業務 — 如果得一盤生意，冇打算擴展到第二個方向，一間公司就夠

🔍 業務低風險 — 如果你嘅生意冇乜訴訟風險（例如：顧問公司、Freelance 平台），資產保護嘅需求唔大

🔍 唔打算賣盤 — 如果你打算一直自己營運到退休，唔打算引入投資者或賣盤，集團架構嘅靈活性優勢用唔著

🔍 公司啱啱起步 — 頭 1-2 年集中火力做好業務先，集團架構係「有規模之後」先考慮嘅嘢

✅ 建議：如果你嘅年收入低於 HK$500 萬，或者業務好簡單（得一兩個客戶），集中火力搞生意好過搞架構。到業務擴展到第二個方向、或者年收入過千萬，先認真考慮集團架構。

9. FAQ 常見問題

控股公司同子公司之間嘅交易要報稅嗎？

要。控股公司同子公司之間嘅交易（例如管理費、貸款利息、服務費）屬於關聯交易，要按公平原則定價，並喺利得稅報稅表中申報。如果交易價格唔合理，稅局有權調整。香港有轉讓定價文件規定，符合特定條件嘅公司要備存轉讓定價文件。

開控股公司要幾多資本？

香港有限公司嘅最低股本係 HK$1，所以理論上開控股公司唔需要好多資本。但實際上，控股公司要有足夠資金去持有子公司股份、支付營運開支（公司秘書費、註冊地址費、報稅費）。建議至少準備 HK$10,000 作為初期營運資金。

子公司嘅利潤點樣上繳俾控股公司？

主要透過派發股息（Dividend）。子公司董事會通過派息決議，將利潤以股息形式派俾股東（即控股公司）。香港冇股息稅，所以控股公司收到股息唔使交稅。控股公司可以將呢啲資金用嚟投資新業務、或者再派息俾你個人。注意：派息要符合公司細則同《公司條例》嘅要求，公司要有足夠嘅可分派利潤（Distributable Profits）。

集團公司嘅審計點樣安排？

每間子公司同控股公司都需要獨立做審計（如果符合審計門檻）。即係如果你有 1 間控股 + 3 間子公司，總共要做 4 份審計報告。審計費會隨公司數量增加。不過，如果係中小型集團，可以搵同一間會計師樓做，通常會俾 discount。另外，如果集團符合特定條件，可以準備「集團賬目」（Group Accounts），展示集團整體財務狀況。

集團公司嘅公司秘書點安排？

每間香港有限公司都要有公司秘書。集團架構下，你可以：(1) 每間公司各自請唔同嘅公司秘書；(2) 由同一間秘書公司處理晒所有公司嘅秘書工作（通常更抵）。好多秘書公司提供「集團套餐」，一次過處理幾間公司嘅周年申報、商業登記續期、會議紀錄等。

海外公司可以做控股公司嗎？

可以。好多香港公司用 BVI（英屬維爾京群島）或者 Cayman Islands 公司做控股公司，特別係打算將來上市或者有國際投資者。BVI 控股公司嘅好處係：零稅率（只要唔喺香港營運）、高度私隱（股東資料唔公開）、彈性高。但要注意：2024 年起 BVI 實施咗經濟實質規定，控股公司都要符合特定申報要求。

📝 總結一句：

集團架構係工具，唔係目標。唔好為咗威或者覺得「專業」就去 set，要真係有商業需要先值得做。如果你有多個業務方向、想保護資產、或者打算將來賣盤，集團架構可以幫你慳稅同慳麻煩。否則，一間公司做好一個業務，其實已經好夠。
