# 📋 香港有限公司公司秘書辭任與更換指南 — 中小企東主必睇

> **一句講晒**：公司秘書辭職唔係佢話走就走咁簡單，公司有法定責任喺限期內搵人頂上。冇公司秘書？公司隨時俾人罰款，嚴重嘅董事仲可能要上庭。

「吓，公司秘書走咗咁大件事㗎？」

我當初就係咁。公司用開嗰間秘書服務公司突然話唔再做我哋生意，send 咗封辭職信嚟就完。我以為「咪再搵過第二間囉」，點知唔知原來法例規定公司秘書一辭職，公司就要喺 **14 日內**通知公司註冊處，而且 **一直都要有名公司秘書**在位。

結果我公司有超過一個月完全冇公司秘書，仲要係我到交 annual return 嗰陣先發現 —— 好彩冇俾人查到，唔係真係俾罰款都似。

---

## 邊啲公司要有公司秘書？

先講基本嘢。根據《公司條例》（第622章），**每間私人有限公司都必須至少有一名公司秘書**。

### 誰可以擔任？
- **個人**：通常居於香港（即係香港地址）
- **公司**：在港有營業地址嘅秘書公司 / 會計師樓

### 最常見嘅做法：
- 你開公司嗰陣間秘書公司自動成為公司秘書
- 或者你自己做（如果你係董事，你可以兼任）
- 或者搵會計師樓 / 秘書公司做，年費幾千蚊

小公司好多都係靠秘書公司，因為慳水慳力。但問題係：如果嗰間秘書公司突然加價、服務差、或者唔撈，你就要硬食。

---

## 公司秘書辭職嘅法定程序

公司秘書辭職唔係佢話句「I quit」就得，佢要：

1. **俾書面辭職信**公司（要正式 letter）
2. **通知公司註冊處**（遞交 `ND2A` 表格 — 公司秘書及董事詳情申報）

但好多時，秘書公司只係 send 封信俾你話佢辭職，佢 **唔會幫你交 ND2A** 㗎！佢辭職信寫明生效日期就拍拍籮柚走人。**你要自己遞交 ND2A 通知公司註冊處。**

### 公司方面嘅法定責任

| 時限 | 要做嘅嘢 |
|------|---------|
| 公司秘書辭職生效日起 | 確保公司仍有一名公司秘書在位 |
| 辭職生效後 **15 日內** | 向公司註冊處遞交 `ND2A` 表格 |
| 如果公司只得一名公司秘書 | 必須喺辭職生效 **前**已任命新秘書（唔可以有 gap） |

如果公司有多過一個秘書（例如你 + 秘書公司），其中一個辭職咁就冇問題 — 另一個頂住。

但如果你得一個秘書，佢一走，公司就 **即刻冇咗公司秘書**。呢個係犯法㗎。

## 冇公司秘書嘅後果（唔好試）

有啲東主覺得「無人秘書公司都 run 到啫」，錯晒。

根據《公司條例》：
- 公司 **必須一直**有公司秘書
- 冇公司秘書嘅日子，每項需要秘書簽署嘅文件都唔合法
- Annual Return（NAR1）要公司秘書簽名，冇秘書簽唔到
- 公司註冊處可以罰公司 **$5,000 - $50,000**
- 嚴重嘅，董事個人可以被檢控

我朋友就中過招：佢間公司秘書走咗冇理，到交 NAR1 嗰陣發現簽名欄要公司秘書簽，佢自己簽咗上去（佢係董事），結果俾公司註冊處打回頭，話董事唔可以簽秘書嗰欄。最後要搵律師樓加急搞返，夾埋駛咗成 $8,000。

## 更換公司秘書 — 步驟教學

### Step 1：搵新公司秘書

視乎你嘅需要：
- **秘書公司（$2,000 - $5,000/年）**：最簡單，包遞交文件、收政府信
- **會計師 / 律師（$5,000 - $15,000/年）**：如果你需要埋會計或法律意見
- **自己做**：如果你係董事，符合資格嘅話可以自己兼任

揀秘書公司要留意：
- 有冇牌照？係咪信得過？
- 有冇 real person 跟你的 case，定係 call centre
- 續約條款：係咪自動續約？有冇 notice period？

### Step 2：新秘書填寫同意書

新嘅公司秘書要簽一份同意書，表示佢願意出任公司秘書。

### Step 3：董事會決議委任

公司需要通過 **董事書面決議** 或 **股東決議** 正式委任新秘書。

記住留低會議記錄或者書面決議 — 之後銀行開戶或做盡職調查可能會問。

### Step 4：遞交 ND2A 表格

遞交 `ND2A - 公司秘書及董事詳情申報》`

需要填：
- 舊秘書嘅辭職日期
- 新秘書嘅姓名 / 名稱
- 新秘書嘅地址
- 新秘書嘅身份證明文件號碼（個人：HKID / 公司：BR 號碼）

遞交方式：
- 🖥️ **網上申請**（eCR）：推薦，S$ 即時完成
- 📄 **紙本提交**：要去金鐘公司註冊處遞交，或者郵寄
- 💰 **費用**：網上 $15，紙本 $30

### Step 5：等公司註冊處更新記錄

如果網上遞交，即時處理。紙本可能要 4-7 個工作日。

之後你嘅公司註冊處記錄就會顯示新秘書資料。

## 公司秘書辭職信嘅關鍵條款（比較）

| 條款 | 好嘅秘書公司 | 唔好嘅秘書公司 |
|------|-------------|--------------|
| Notice period | 1-3 個月通知 | 即時辭職 |
| 協助交接 | 有專人交接文件 | 掉低就走 |
| 轉介新秘書 | 會推薦同行 | 乜都唔理 |
| 已簽未交文件 | 全部交返俾你 | 唔會通知你 |

簽約前一定要睇清楚 terminate 條款！我見過有秘書公司要 3 個月 notice，就算你對佢唔滿意都要焗住俾多 3 個月錢。

## 特別情況處理

### 情況 1：秘書公司突然執笠
- 法理上秘書已經辭職（因為公司唔存在）
- 你必須盡快搵新秘書 + 遞交 ND2A
- 就算你冇收到辭職信，都要主動處理
- **建議**：立即 email 通知公司註冊處，留低記錄

### 情況 2：公司秘書係海外人（唔係香港居民）
根據《公司條例》：
- 如果公司秘書係個人，必須「通常居於香港」
- 如果你嘅秘書移民走咗，佢嘅辭職即刻生效
- 你要遞交 ND2A 申報佢辭職，同時委任新秘書

### 情況 3：公司秘書係公司（另一間有限公司）
- 如果秘書公司唔做，佢嘅 director 簽辭職信就得
- 你照樣要遞交 ND2A，申報「該公司不再擔任秘書」
- 新秘書可以係個人或者另一間公司

## 慳錢貼士

| 做法 | 大約費用 | 適合 |
|------|---------|------|
| 自己做公司秘書 | $0（如你符合資格） | 公司好少文件要搞 |
| 平價秘書公司（純行政） | $1,500 - $3,000/年 | 中小企，有基本文件 |
| 全套秘書 + 註冊地址 | $3,000 - $6,000/年 | 冇實體辦公室嘅公司 |
| 會計師樓包秘書 | $6,000 - $15,000/年 | 需要會計 + 審計 + 秘書一條龍 |

我自己而家用緊一間 $2,500/年嘅秘書公司，包遞交文件、收政府信、提醒 annual return 到期。對我嚟講夠晒用。

但係弊處係：佢哋人工低，跟進速度慢，有 urgent 嘢要自己追。如果間公司交易多、文件複雜，就建議俾多少少錢搵好啲嘅。

## 常見錯誤（我中過嘅）

### ❌ 錯誤 1：冇留意秘書公司條款
我以為秘書公司「一定幫你做晒所有嘢」，點知原來 ND2A 要自己交。

**正確**：簽約前問清楚：
- 續約係咪自動？
- 辭職 notice period 幾耐？
- 佢辭職時會唔會幫我交 ND2A？

### ❌ 錯誤 2：秘書走咗冇即時搵人
我嗰陣諗住「一個月先至搵都得掛」，點知期間要簽嘅文件全部簽唔到，夾埋 delay 咗成個 project。

**正確**：秘書一俾 notice，即刻開始搵新秘書。最好兩星期內交接完。

### ❌ 錯誤 3：以為董事可以代替秘書簽名
Annual return 嗰個簽名欄係特定俾秘書簽嘅。董事簽名係另一欄。唔可以交叉簽。

**正確**：公司秘書簽公司秘書嗰欄，董事簽董事嗰欄。冇秘書嘅話，啲文件就簽唔到。

### ❌ 錯誤 4：冇保留董事會決議記錄
有次銀行要求睇「秘書委任記錄」，我摷唔到文件，搞到要補做一份 — 又要董事再簽一次名。

**正確**：每次委任 / 辭職秘書，都要有書面記錄。放喺公司 minute book 入面。

## 總結

公司秘書辭職係好 common 嘅事，特別係而家好多秘書公司服務質素參差。最緊要記住：

1. ✅ 公司**必須一直**有公司秘書在位
2. ✅ 秘書辭職後 **15 日內**要遞交 ND2A
3. ✅ 如果得一個秘書，必須先搵人先好俾佢走
4. ✅ 揀新秘書公司要睇條款，特別係 notice period
5. ✅ 每次變更都要留記錄 — 董事決議 + ND2A

公司秘書呢樣嘢，你唔理佢就冇事，一旦有事就好大件事。最緊要係知道程序係點行，然後搵個可靠嘅合作伙伴。

你而家有冇公司秘書？如果係秘書公司做緊，建議你而家摷返份合約出嚟睇下 notice period — 唔使等出事先嚟後悔。
