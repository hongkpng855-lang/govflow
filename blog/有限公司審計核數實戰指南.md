# 📊 香港有限公司審計（核數）實戰指南 — 中小企東主第一次做 Audit 必讀

> **一句講晒**：每年收到 audited accounts 嗰叠數就頭痕？唔知點解一定要做審計？俾會計師樓收費嚇親？由親身經歷過嘅東主話你知審計係咩一回事，點樣準備先唔使俾人劏。

「我間公司啱啱開，年年都冇賺錢，點解仲要俾幾千蚊做審計？」

呢句說話，我第一年做東主嘅時候都講過。當年我仲諗住「生意唔好就唔使報稅啦」，結果收到稅局封信先知道中伏。

**事實係**：香港《公司條例》規定，所有有限公司每年都要委任核數師審計財務報表，就算公司冇營業額、冇賺錢、甚至係零收入，都要照做。

唔係嚇你，唔做審計嘅後果：
- 公司註冊處罰款（最高 **HK$300,000**）
- 董事可以被檢控
- 銀行戶口可能被凍結
- 將來賣盤／融資完全唔使諗

我第一年就係因為唔識，俾間會計師樓收咗 **$8,000** 做審計，後來先知可以平好多。等我分享下呢幾年嘅經驗，幫你慳返唔少錢同時間。

---

## 🔍 邊啲公司需要做審計？

先搞清楚自己公司駛唔駛做。

### ✅ 一定要做嘅情況
- **所有香港有限公司**，無論規模
- 即使：零收入、零交易、休眠公司（Dormant Company）
- 除非：向公司註冊處申請咗「休眠狀態」並獲批

### ❌ 唔使做嘅情況
- 獨資經營（Sole Proprietorship）— 報利得稅就得
- 合夥業務（Partnership）— 報稅就得，唔使审计
- 公司已正式解散或撤銷註冊

### 💡 休眠公司例外
如果你間公司真係完全冇營運（冇收入、冇開支、冇銀行出入），可以申請做 **Dormant Company**，咁就可以豁免審計。但係申請手續都要做，唔係自動嘅。

---

## 📋 審計要做啲咩？（逐步拆解）

審計唔係你想像中咁恐怖，其實係核數師幫你 check 盤數係咪真確。

### Step 1：準備財務報表（Financial Statements）

審計之前，你要先有：
- **損益表（Profit & Loss）**
- **資產負債表（Balance Sheet）**
- **現金流量表（Cash Flow Statement）**
- **財務報表附註（Notes to Accounts）**

多數中小企都係由會計師幫你執，但你至少要俾齊以下文件：

### Step 2：準備 supporting documents

呢啲係核數師一定會問你拎嘅：

| 文件 | 用途 | 注意 |
|------|------|------|
| 銀行月結單 | 核對所有出入數 | 全年每份都要保留 |
| 銷售發票 + 合約 | 證明收入真確 | 要同銀行入數對得上 |
| 採購單 + 收據 | 證明開支真實 | 冇單據嘅開支可能被 disallow |
| MPF 供款紀錄 | 核對員工薪酬 | 年度報表要留底 |
| 董事/股東決議 | 重要決定紀錄 | 特別係借貸或資產買賣 |
| 租約 / 水電費單 | 證明營運地址 | 地址要同商業登記一致 |
| 銀行貸款文件 | 確認負債 | 利息支出核對 |

### Step 3：核數師審閱

核數師會做以下嘢：
1. 抽查你的交易（抽樣檢查）
2. 寄信俾銀行確認戶口結餘（銀行確認信）
3. 核對存貨（如果有）
4. 檢查內部控制
5. 出報告（Audit Report）

### Step 4：出審計報告（Audit Report）

核數師會俾一個意見：
- **Unqualified Opinion（無保留意見）**— 正正常常，最理想
- **Qualified Opinion（保留意見）**— 有啲問題但唔嚴重
- **Disclaimer（不發表意見）**— 資料唔夠，出唔到意見
- **Adverse（否定意見）**— 盤數有問題

中小企九成都係出第一種，只要你文件齊全就冇問題。

## 💰 審計收費：點樣先唔會俾人劏？

呢個係最多人問嘅問題。我第一年俾人收 **$8,000**，後來轉咗另一間收 **$3,500**。

### 影響收費嘅因素

| 因素 | 平 | 貴 |
|------|-----|-----|
| 公司規模 | 冇交易 / 好少交易 | 多供應商 + 多客戶 |
| 交易數量 | < 50 筆/月 | > 200 筆/月 |
| 文件整齊度 | 全部 Excel 整理好 | 散件、冇分類 |
| 有冇存貨 | 服務業冇存貨 | 貿易公司有存貨 |
| 揀咩核數師 | 小型會計師行 | Big 4 / 中型行 |

### 市場參考價（2026 年）

- **休眠公司**：HK$2,000 - $3,000
- **小型公司（< 50 筆交易）**：HK$3,500 - $6,000
- **中型公司（50-200 筆）**：HK$6,000 - $12,000
- **貿易公司（有存貨）**：HK$8,000 - $15,000

### 慳錢貼士

1. **文件整理好先交** — 核數師慳時間就係慳你錢
2. **用會計軟件** — Xero / QuickBooks / 波鞋會計軟件，減少人手執數時間
3. **年中問定 quote** — 每年 3-4 月係高峰，會貴啲
4. **同一間做 Audit + Tax** — 通常有 discount
5. **問清楚包唔包 Tax Return** — 好多報價只包 Audit，報稅另計

## 🚩 5 個常見陷阱（我中過至少 3 個）

### 🚩 陷阱一：冇保留單據就亂估

「呢張單唔見咗，我估大約 $5,000 啦得唔得？」

**唔得。** 核數師要有原始單據先可以確認開支。冇單嘅開支，核數師會要求你 disallow（剔除），即係要補稅。

我試過有一次唔見咗張 $8,000 嘅顧問費單據，結果核數師話要 disallow，嗰年要因為呢 $8,000 補番約 $1,300 稅。

**解決方法**：所有單據 scan 存檔，Google Drive / Dropbox 放一份。

---

### 🚩 陷阱二：以為董事貸款冇問題

「我借咗 $50,000 俾公司周轉，有咩問題？」

**好大問題。** 董事貸款係核數師一定會 chur 嘅位。如果你冇正式借貸協議、冇利息條款、冇還款時間表，核數師可能會要求你將筆數 reclass 做「股本投入」或者「股東往來」，影響盤數。

仲有，如果你借錢俾公司而收利息，稅局可能當係你的個人收入，要報薪俸稅。

**解決方法**：所有董事貸款一定要有書面協議，寫明還款期同利息。

---

### 🚩 陷阱三：遺忘咗未收賬（Accruals）

「呢張單 $20,000 係 3 月出嘅，但係客 4 月先找數，咁即係下年先入數啦？」

**錯。** 權責發生制（Accrual Basis）要求你將收入同開支記錄喺發生嘅年度，而唔係 cash 到賬嘅年度。

核數師會幫你調整，但調整愈多，審計費愈貴。

---

### 🚩 陷阱四：MPF 記錄亂晒龍

「我公司得 2 個員工，MPF 求其交就算啦。」

核數師一定會對 MPF 紀錄同薪酬係咪一致。如果你請咗人但冇供 MPF（犯法㗎），或者供錯數，核數師係有責任喺報告度披露嘅。

我見過有間公司因為 MPF 紀錄亂，核數師要出 Qualified Opinion，之後銀行唔肯借錢俾佢。

---

### 🚩 陷阱五：自己不懂裝懂，到死線先找人做

「審計啫，最後一刻先搵會計師樓啦。」

最大嘅錯誤。審計係要時間嘅——核數師要排期、要等銀行 confirmation letter、要 review。如果你到 3 月先搵人做前一年嘅審計，高峰期好多行家都唔接，或者收你 rush fee（加價 50%）。

**最佳時間**：每年 6-9 月（公司財政年度結束後 3-6 個月內），避開高峰期（11 月至 3 月）。

---

## ✅ 年度審計 Checklist

搞掂審計其實唔難，最緊要有系統。每年跟呢個 checklist 就得：

- [ ] 確定財政年度結束日期（多數公司用 12 月 31 日 / 3 月 31 日）
- [ ] 全年銀行月結單整理好
- [ ] 全年發票 + 收據 + 合約分類存檔
- [ ] MPF 年度報表對數
- [ ] 盤點存貨（貿易公司適用）
- [ ] 整理董事貸款記錄
- [ ] 整理股東名冊變動
- [ ] 預約核數師（建議年度結束後 3 個月內）
- [ ] 核數師出 Audit Report + Tax Return
- [ ] 簽署文件，核數師交稅局 + 公司註冊處
- [ ] 保留文件最少 7 年（公司法規定）

---

## 💁 最後想講嘅嘢

審計聽落好恐怖，但其實只要你平時做好記錄，每年搵個可靠嘅核數師，成個過程係好順暢嘅。

最緊要唔好做最後一刻先開始準備。我第一年係死 line 前兩星期先開始搵人，結果俾人收 rush fee 之餘，仲要自己執餐死。

如果你係第一次做審計，建議你：
1. 今年年中就開始問 quote
2. 搵個細規模但 responsive 嘅會計師行（唔使 Big 4）
3. 問清楚報價包咩（Audit only？定包 Tax Return？）
4. 文件整理好先俾人

**審計係有限公司嘅必要代價，但唔代表要俾人劏。**
