1. 前言：香港公司跨境交易點解咁重要

香港作為國際金融中心，大部分有限公司都需要處理跨境交易——無論係向內地供應商採購、收海外客戶美金、定係同集團公司之間資金調撥。但跨境交易唔係就咁轉帳就得，入面有好多法律、稅務同合規陷阱。

我公司第一年就試過因為唔識處理跨境匯款，筆錢俾銀行hold咗成個星期，因為冇準備齊貿易文件。之後仲試過俾稅局問關聯交易定價，好彩最後解釋得通，但已經嚇餐死。如果你都係第一次搞跨境交易，呢篇文幫你避開晒啲伏。

📌 一句講晒：香港冇外匯管制，但銀行有KYC要求，收款國有外匯管制，稅局有關聯交易審查。跨境交易唔係就咁T/T就得，文件準備同合規安排先係關鍵。

2. 常見跨境交易類型

香港公司跨境交易主要分以下幾類，每類嘅文件同合規要求都唔同。

2.1 貨物貿易（Import / Export）

最常見嘅跨境交易。交易文件包括：

  • Commercial Invoice — 列明貨物描述、數量、單價、總額、貿易條款（Incoterms）
  • Packing List — 包裝細節、件數、重量
  • Bill of Lading / Air Waybill — 提單/空運提單
  • 合同 / Sales Contract / Purchase Order — 雙方簽署嘅買賣合約
  • 產地來源證（CO） — 如果需要申請關稅優惠

銀行匯款時通常要求提供上述文件，尤其係第一次交易或者大額匯款。

2.2 服務貿易（Service / Professional Fees）

包括顧問費、設計費、IT服務費、市場推廣費等。文件要求：

  • Service Agreement — 服務合約，列明服務範圍、收費方式、付款條款
  • Invoice — 服務費發票
  • Statement of Work（SOW） — 工作範圍說明
  • Completion Report — 完成報告（尤其大額交易）

💡 貼士：服務貿易係稅局關注嘅重點，尤其係跨境關聯方服務費。要確保服務確實有提供，收費合理，否則稅局有權調整甚至罰款。

2.3 關聯方交易（Related Party Transactions）

同集團公司、母公司、子公司、或有關連嘅個人之間嘅交易。常見包括：

  • 管理費（Management Fees）
  • 特許權使用費（Royalty / License Fee）
  • 貸款利息（Intercompany Loan Interest）
  • 貨物及服務交易
  • 成本分攤（Cost Sharing / Cost Contribution）

⚠️ 高度注意：關聯方交易係稅局審查重點。所有關聯交易都要按公平原則定價（Arm's Length Principle），並備有轉讓定價文檔。詳情見第5節。

2.4 資金調撥（Fund Transfer / Capital Injection）

包括股東貸款還款、股息分派、增資注資、集團內部資金池等。文件要求：

  • 貸款協議 / Loan Agreement
  • 董事會決議 / Board Resolution（授權交易）
  • 還款時間表
  • 股息分派：審計報告 + 分紅決議
  • 增資注資：股份配發文件 + 公司註冊處申報（NSC1）

3. 資金匯出匯入實戰

好多香港公司東主以為開咗公司戶口就可以任轉錢，但實際上銀行對跨境匯款有嚴格嘅KYC同AML審查。以下係實戰經驗：

3.1 匯款前準備

  • 了解銀行嘅要求 — 每間銀行嘅KYC要求唔同，HSBC、中銀、渣打各有自己嘅表格同文件要求
  • 準備貿易文件 — 即使係細額匯款（幾萬蚊），銀行都有權要求睇文件
  • 申報交易性質 — 要清楚填寫匯款用途，例如「貿易付款」、「服務費」、「股東貸款還款」
  • SWIFT Code 同 IBAN — 確認收款銀行嘅SWIFT Code正確，錯一個字母就彈返轉頭

3.2 大額匯款注意事項

💡 實戰經驗：一般嚟講，單筆超過HK$50萬（或等值外幣）嘅跨境匯款，銀行多數會問背景。超過HK$100萬嘅，幾乎一定會要求提供合約同發票。我試過轉HK$200萬俾內地供應商，銀行hold咗3日，最後要補返成疊文件先放行。

3.3 常用跨境匯款渠道比較

渠道速度手續費匯率適合
傳統銀行電匯（T/T）1-3工作日HK$150-300銀行匯率（較差）大額、正式交易
Wise / Airwallex / OFX即時-1日0.4-1%市場中間價中小額、快捷
虛擬銀行（ZA / Livi / Mox）1-2日HK$0-100中等細額、個人使用
加密貨幣（USDT/USDC）即時低市場價極細額、需合規

3.4 常見匯款用途代碼

填匯款申請表時，要揀啱用途代碼（Transaction Purpose Code），最常見嘅有：

  • 101 — 貨物貿易付款
  • 201 — 服務貿易付款
  • 301 — 直接投資（如注資、股東貸款）
  • 401 — 證券投資
  • 501 — 貸款還款
  • 601 — 股息/利潤匯出

4. 中國外匯管制攻略

如果你嘅跨境交易涉及中國大陸（香港公司最大嘅貿易伙伴），一定要了解中國外匯管制制度。以下係關鍵要點：

4.1 中國外匯管制基本規則

  • 中國實行嚴格外匯管制，人民幣不可自由兌換
  • 所有跨境資金進出要有真實交易背景（真實性審核）
  • 貨物貿易：要提供報關單、合同、發票
  • 服務貿易：要提供合同、發票、稅務備案
  • 直接投資：要經商務部/發改委審批或備案

4.2 香港公司收人民幣（跨境貿易人民幣結算）

香港係最大嘅離岸人民幣中心，收人民幣唔係問題，但要留意：

  • 香港銀行接受人民幣存款，但匯率同大陸有差價（CNH vs CNY）
  • 跨境貿易人民幣結算要符合真實交易背景
  • 人民幣資金可以自由喺香港兌換成港幣或外幣
  • 如果想將人民幣匯返大陸，要有貿易背景

4.3 內地公司匯款來港

如果你嘅內地關聯公司或客戶需要匯款來香港，要留意：

  • 經常項目（貿易、服務費）— 憑合同、發票、稅務備案表可匯出
  • 資本項目（投資、貸款）— 需要外管局審批或FDI備案
  • 利潤匯出 — 要先完稅（繳納企業所得稅），再經審計確認可分配利潤
  • 個人結匯 — 內地居民每年5萬美元結匯限額

⚠️ 常見陷阱：唔好試圖用地下錢莊或虛假貿易方式將資金調出中國。近年中國嚴厲打擊地下錢莊同虛假貿易，一經發現可能面臨刑事責任。一定要用合法渠道。

4.4 香港公司喺大陸開設辦事處/子公司

香港公司如果想喺大陸經營，有以下選擇：

  • 代表處（Rep Office） — 只能從事聯絡活動，唔可以有營業收入
  • 外商投資企業（WFOE） — 可以實際經營，需要商務部備案
  • 合資企業（JV） — 同內地合作方成立，要章程同合資協議

成立WFOE後，資金進出要透過註冊資本（Capital Injection）同經常項目（如服務費、特許權費）方式進行。

5. 轉讓定價（Transfer Pricing）合規

轉讓定價係指關聯方之間交易嘅定價安排。香港2018年開始正式實施轉讓定價條例（《稅務條例》第50A條），所有關聯交易都要符合公平原則。

5.1 邊啲公司要留意轉讓定價？

  • 有關聯方跨境交易嘅香港公司
  • 集團內部有管理費、特許權費、利息收支
  • 香港公司同海外關聯公司有貨物/服務交易
  • 利潤異常低或持續虧損但母公司賺錢

5.2 轉讓定價文檔要求

根據稅局指引，符合以下條件嘅公司需要備有轉讓定價文檔：

文檔類型適用條件內容
主體檔案（Master File）集團總營業額≥HK$7.5億集團業務架構、全球價值鏈、無形資產政策
本地檔案（Local File）關聯交易超過門檻：貨物≥HK$2.2億；服務/其他≥HK$4,400萬關聯交易詳情、定價政策、功能風險分析、可比分析
國別報告（CbCR）集團總營業額≥HK$68億各國收入、利潤、稅款分配

💡 中小企貼士：就算你嘅公司未達到上述門檻，稅局仍然有權要求你解釋關聯交易定價。建議至少要備有：關聯交易清單、定價政策說明、支持定價嘅市場分析或可比數據。尤其係首年有跨境關聯交易，更加要準備好。

5.3 公平原則（Arm's Length Principle）

簡單嚟講，關聯交易嘅定價要同非關聯方之間嘅交易價格一致。常用嘅定價方法包括：

  • 可比非受控價格法（CUP） — 參考同類產品/服務嘅市場價格
  • 成本加成法（Cost Plus） — 成本 + 合理利潤率
  • 再銷售價格法（Resale Price） — 最終售價減合理利潤
  • 交易淨利潤法（TNMM） — 參考同行嘅利潤率水平
  • 利潤分割法（PSM） — 按貢獻比例分配利潤

6. 雙重課稅安排

香港與超過40個國家/地區簽訂咗全面性避免雙重課稅協定（CDTA），包括中國內地、日本、新加坡、英國、馬來西亞等。利用好稅務協定可以幫你慳返唔少稅。

6.1 股息預扣稅優惠

香港公司收取海外子公司股息時，如果當地有同香港簽訂稅務協定，預扣稅率可能大幅降低：

  • 中國內地：一般10% → 協定後5%（持股≥25%）
  • 日本：一般20% → 協定後5%（持股≥25%）
  • 新加坡：一般0%（新加坡唔徵收股息預扣稅）
  • 英國：一般0%（英國唔徵收股息預扣稅）

6.2 特許權費/利息預扣稅

  • 中國內地特許權費：一般10% → 協定後7%
  • 中國內地利息：一般10% → 協定後7%（銀行貸款除外）
  • 要享受協定優惠，需要向稅局申請「居民證明書」（COR）

📌 關鍵提醒：要享受稅務協定優惠，必須確保香港公司有經濟實質（Economic Substance）——即係有實際辦公室、員工、做實際決策。如果只係空殼公司，稅局可以拒絕發出COR，或者對方國家拒絕給予協定優惠。呢點近年在國際稅務環境下越嚟越重要。

7. 必備文件清單

無論係邊種跨境交易，以下文件都建議準備齊全：

文件用途備註
交易合約 / Agreement證明交易背景雙方簽署，列明價格、付款條款、交貨條件
商業發票 / Commercial Invoice匯款依據同合約一致
銀行匯款單 / SWIFT MT103匯款憑證保留銀行收據
董事會決議 / Board Resolution授權大額交易尤其關聯交易、貸款、投資決策
稅務申報文件報稅用轉讓定價文檔、COR申請
審計報告利潤匯出/分紅審計後確認可分配利潤
運單 / 報關記錄貨物貿易海關申報記錄

8. 7 個常見陷阱

❌ 陷阱 1：冇準備文件就匯款
銀行hold住筆錢成個星期係小事，最慘係交易告吹。解決方法：匯款前先問銀行要咩文件，準備好先submit。

❌ 陷阱 2：關聯交易冇定價政策
稅局查帳時會逐筆關聯交易問點解呢個定價。冇合理解釋嘅話，稅局有權重新調整利潤，補稅加罰款。

❌ 陷阱 3：以為香港冇外匯管制就咩都得
香港雖然冇外匯管制，但銀行有KYC責任。如果你嘅資金來源唔清唔楚，銀行可以凍結戶口甚至關閉戶口。

❌ 陷阱 4：跨境服務費冇做Withholding Tax
如果香港公司支付服務費俾境外公司，要留意當地有冇徵收預扣稅，尤其係特許權費、技術服務費。

❌ 陷阱 5：跨境貸款協議冇寫清楚
股東貸款俾公司或公司貸款俾附屬公司，一定要有書面貸款協議，列明利率、還款期、擔保條款。否則稅局可以質疑係咪真係貸款，定係股本投入。

❌ 陷阱 6：冇考慮匯率風險
跨境交易通常涉及外幣，匯率波動可以好大。尤其係長期合約，建議用遠期合約鎖定匯率，或者喺合約入面寫明匯率調整機制。

❌ 陷阱 7：用地下錢莊調資金
為咗慳手續費或者避開外匯管制，有人會用地下錢莊。呢個風險極高——唔單止違法，仲可能被騙，近年香港同內地聯手打擊，一經發現會面臨刑事檢控。

9. 常見問題（FAQ）

香港有限公司跨境匯款有冇金額限制？

香港冇法定外匯管制，所以冇法定金額上限。但銀行會按KYC政策進行審查，一般嚟講：

  • HK$10萬以下：通常唔會點問
  • HK$10萬-HK$50萬：可能有簡單問題
  • HK$50萬以上：好大機會要求提供文件
  • HK$100萬以上：幾乎一定會要求齊全嘅貿易文件

香港公司點樣匯款去大陸最方便？

最常見嘅方法：

  • 直接銀行電匯（T/T） — 匯港幣俾大陸供應商，對方自行兌換人民幣
  • 跨境人民幣結算 — 直接匯人民幣，但要雙方銀行都有跨境人民幣結算服務
  • Wise / Airwallex — 用FinTech平台匯款，匯率好、速度快、透明度高
  • 透過香港中資銀行 — 中銀香港、工銀亞洲等有內地關聯銀行，匯款更方便

跨境交易嘅文件要keep幾耐？

根據《公司條例》同《稅務條例》，跨境交易相關文件至少保存 7年。建議：

  • 交易合約、發票、銀行記錄 — 7年
  • 轉讓定價文檔 — 7年（由相關課稅年度完結起計）
  • 公司決議、股權記錄 — 永久保存

跨境服務費使費可以扣稅嗎？

可以，但要符合以下條件：

  • 該服務確實用於產生應評稅利潤
  • 費用係實際發生嘅（唔係虛構）
  • 金額合理（符合公平原則）
  • 有足夠文件證明（合約、發票、付款記錄）

香港公司可以收加密貨幣作為跨境交易付款嗎？

可以，但要留意：

  • 香港冇禁止公司收加密貨幣，但銀行戶口未必支援
  • 收到加密貨幣後，要兌換返法幣先入到銀行戶口（除非用加密貨幣友善銀行）
  • 稅務上，加密貨幣被視為資產或財產，交易要按市值計價
  • 要保留完整嘅交易記錄供審計同報稅用
  • 建議先用傳統貨幣做跨境交易，加密貨幣等熟悉咗先考慮

點樣申請香港稅務居民證明書（COR）？

申請COR係享受稅務協定優惠嘅第一步：

  • 向香港稅務局提交IR1313A表格（個別國家有專用表格）
  • 證明公司在香港有實際經營（辦公室、員工、決策）
  • 提供公司註冊證書、商業登記證、審計報告
  • 處理時間約2-4星期
  • COR有效期通常係一個課稅年度
