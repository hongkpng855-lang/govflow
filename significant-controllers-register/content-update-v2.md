## 微調 1：Step 2 — 共同安排（Joint Arrangements）正式用詞

**依據：** CR Guideline §11.2 — 引入「共同安排」法理，指明一致行動人士必須合計持股。

### 1a. Step 2 `details` — 測試 1 用詞

**檔案：** `scr-processes.json` → step[1]（index 1, stepNumber=2）→ `details`

**原文：**
```
直接或間接持有公司多過 25% 嘅股份。計算時要包括關連人士嘅持股。Nominee 持有嘅股份視為受益人持有。如公司冇股本，則有權分佔超過 25% 資本或利潤。
```

**新文：**
```
直接或間接持有公司多過 25% 嘅股份。計算時要包括關連人士嘅持股，以及根據共同安排（Joint Arrangements）一致行動嘅人士之合計持股。Nominee 持有嘅股份視為受益人持有。如公司冇股本，則有權分佔超過 25% 資本或利潤。
```

**原文：**
```
直接或間接控制多過 25% 嘅投票權。有啲股東持股少但投票權多，要分開計。
```

**新文：**
```
直接或間接控制多過 25% 嘅投票權。計算時須合計共同安排（Joint Arrangements）下一致行動人士嘅投票權。有啲股東持股少但投票權多，要分開計。
```

## 微調 3：Step 2 — 間接持股穿透 & 多數利益鏈定義（§11.5）

**依據：** CR Guideline §11.5 — 間接持股必須沿「多數利益鏈」（>50% 投票權/控制權鏈）穿透，否則不視為控制。

### 3a. Step 2 `details` — 測試 1 補充

喺測試 1「直接或間接持有公司多過 25% 嘅股份」之後插入新段落：

**插入位置：** 測試 1 描述結尾（「如公司冇股本，則有權分佔超過 25% 資本或利潤。」之後），「📋 測試 2：」之前

**新插入文字：**
```
⛓️ 間接持股穿透規則（§11.5）：
如果股東係透過一間或多間中間實體間接持股，必須沿「多數利益鏈」（Majority Interest Chain）穿透計算——即每一層中間實體嘅投票權或控制權必須 >50%，才能計入最終持有人嘅持股比例。若某層控制權 ≤50%，該鏈條即中斷，無法穿透。
```

### 3b. Step 2 `details` — ⚠️ 唔好做嘅事 section

**原文 bullet：**
```
• 唔好只睇表面持股 → 間接持股（透過其他公司持有）都要計
```

**新文：**
```
• 唔好只睇表面持股 → 間接持股（透過其他公司持有）都要計，但必須符合多數利益鏈（>50% 控制權）方可穿透
```

### 3c. Step 2 `details` — ✅ 一定要做 section

**原文 bullet：**
```
• 如有間接持股或法團股東，穿透到最終自然人；Nominee 持股須穿透至實益擁有人
```

**新文：**
```
• 如有間接持股或法團股東，沿多數利益鏈（>50% 控制權）穿透到最終自然人；Nominee 持股須穿透至實益擁有人
```

## 微調 4：Step 6 — 控制性質字眼改用 Annex E 訂明措辭

**依據：** CR Guideline Annex E — 控制性質必須使用 Annex E 嘅法定措辭，唔可以用自訂描述。

### 4a. Step 6 `details` — 控制性質欄位說明

**檔案：** `scr-processes.json` → step[5]（index 5, stepNumber=6）→ `details`

**原文（📋 SCR 必須包含嘅資料欄位 第 5 點）：**
```
5. 控制性質（符合邊項測試條件）
```

**新文：**
```
5. 控制性質（必須使用 Annex E 訂明措辭，見下方對照表）
```

### 4b. Step 6 `details` — ⚠️ 唔好做嘅事 section

**原文 bullet：**
```
• 控制性質唔好寫得太模糊（就寫「股東」而唔係「持有超過25%股份」）
```

**新文：**
```
• 控制性質一定要用 Annex E 訂明措辭，唔可以用自訂描述
```

### 4c. Step 6 `details` — ✅ 一定要做 section

**原文 bullet：**
```
• 控制性質要對應 Step 2 嘅測試結果
```

**新文：**
```
• 控制性質要對應 Step 2 嘅測試結果，並使用 Annex E 訂明措辭：
   條件 (a)：該人直接或間接持有公司 25% 以上的已發行股份。
   條件 (b)：該人直接或間接持有公司 25% 以上的表決權。
   條件 (c)：該人直接或間接有權委任或罷免公司董事會的過半數董事。
   條件 (d)：該人有權對公司行使或實際行使重大影響或控制。
   條件 (e)：該人對信託或商號有權行使或實際行使重大影響或控制，而該信託或商號的受託人或成員符合 (a)-(d) 中任何條件。
```

