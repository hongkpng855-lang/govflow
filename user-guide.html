ESGov 溝通指南 用箭嘴叫區域 · 講詞彙

🗺️ 點樣同 builder/SEO 溝通

下面每張 screenshot 都有箭嘴指住每個區域，寫明「呢度係 XXX · 我叫佢做【詞彙】」。你以後就直接用 粗體詞彙 同我講就得。

📖 常用詞彙

產品一

主頁第一個產品 card：
股份轉讓完整指南
URL: /shareholder-transfer

產品二

主頁第二個產品 card：
更改名稱完整指南
URL: /company-name-change

產品三

主頁第三個產品 card / SCR 產品：
重要控制人登記冊 (SCR)
URL: /significant-controllers-register

Step N

SCR 產品頁嘅第 N 個步驟（1-7）。
例：「Step 3 嘅 document card 張圖」

Generator

可填寫 + 生成 PDF 嘅工具頁面。
例：「/scr-notice-generator 嘅表頭」

Document Card

SCR 頁面入面每份文件嘅 card，頂部有 demo 圖片、下面係 title + 描述 + 制。

Blog card

主頁或 Blog listing 嘅文章 card：
emoji + title + 簡述 + date

demoImage

SCR document card 嘅頂部預覽圖。有 Generator 嘅步驟先有。

準備中

未有真實範例／內容嘅 placeholder。顯示 🛠️ 準備中 badge。
